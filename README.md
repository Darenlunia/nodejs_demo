# nodejs_demo
是一个很小的demo，走了一下使用js完成服务器操作的流程，借助nodejs环境完成http的get和post请求。

#### 运行方法
官网安装nodejs环境,打开app.js所在文件夹

```
执行node app
访问localhost:8888/
```
#### 两个接口
只有登录和注册两个接口，功能和页面比较简陋，但简明的完成了nodejs执行服务器执行http操作的基本流程。
###### 1 ./login 
参数 username password
返回值 err msg
###### 2 ./reg
参数 username password
返回值 err msg
